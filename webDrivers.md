Con gusto te ayudaré a crear una guía detallada, paso a paso, diseñada para una instalación limpia de Windows de 64 bits.Aquí tienes el artículo completo, con todos los pasos para configurar tu entorno de automatización web con KNIME, cubriendo tanto Chrome como Edge.⚙️ Guía Definitiva: Configuración de WebDrivers para KNIME en Windows (Chrome & Edge)Un Problema Común: "Unable to obtain driver"Si estás inmerso en la automatización web con KNIME, tarde o temprano te encontrarás con el temido error "Unable to obtain driver for Edge/Chrome...".Este problema significa que KNIME no puede encontrar o ejecutar el programa que le permite "hablar" con tu navegador, conocido como WebDriver. Solucionarlo es un proceso de tres pasos: Coincidencia de Versiones, Descarga y Configuración del PATH de Windows.Esta guía asume que tienes una instalación limpia de Windows 64-bit y te guiará para tener ambos WebDrivers listos para KNIME.1. Coincidencia de Versiones: El Primer Paso CríticoEl error más común es usar un WebDriver que no coincide con la versión de tu navegador. Los tres primeros números de la versión deben coincidir.1.1. Verificar la Versión de Tu NavegadorNavegadorCómo verificar la versiónGoogle ChromeAbre Chrome, escribe chrome://version en la barra de direcciones y presiona Enter.Microsoft EdgeAbre Edge, escribe edge://version en la barra de direcciones y presiona Enter.Ejemplo de versión: Si tu versión de Chrome es 129.0.6647.110, necesitas un ChromeDriver que comience con 129.0.6647.2. Descarga de los WebDriversUna vez que sepas las versiones exactas, procede a la descarga.2.1. ChromeDriver (Para Google Chrome)Visita el sitio oficial: Ve a la página de descargas de ChromeDriver.Busca la versión: Localiza la versión del driver que coincida con los primeros tres números de la versión de tu Chrome.Descarga: Haz clic en el enlace de la versión y luego selecciona el archivo ZIP para tu arquitectura (win64).Descomprime: Extrae el archivo ZIP. Encontrarás el ejecutable chromedriver.exe.2.2. MSEdgeDriver (Para Microsoft Edge)Visita el sitio oficial: Ve a la página de descargas de MSEdgeDriver (Microsoft Edge WebDriver).Busca la versión: Al igual que con Chrome, localiza el enlace que corresponda a los primeros tres números de tu versión de Edge.Descarga: Haz clic en el enlace para descargar el archivo ZIP.Descomprime: Extrae el archivo ZIP. Encontrarás el ejecutable msedgedriver.exe.3. Configuración del PATH de Windows (Solución Global)El PATH es una lista de directorios donde Windows buscará automáticamente los ejecutables. Esta es la forma más limpia de que KNIME (y cualquier otro programa) encuentre los drivers sin configuración adicional.3.1. Crear la Carpeta de DriversCrea una carpeta dedicada en la raíz de tu disco C: para guardar los WebDrivers de forma organizada.Ruta recomendada: C:\WebDriversCopia los archivos ejecutables que descargaste y descomprimiste a esta nueva carpeta:C:\WebDrivers\chromedriver.exeC:\WebDrivers\msedgedriver.exe3.2. Modificar la Variable de Entorno PATHAbre el menú de Inicio y busca "Editar las variables de entorno del sistema" (o "Variables de entorno").En la ventana de Propiedades del sistema, haz clic en el botón "Variables de entorno...".En la sección "Variables del sistema" (la de abajo), busca y selecciona la variable llamada Path.Haz clic en "Editar...".En la ventana de edición, haz clic en "Nuevo" y añade la ruta de tu carpeta de drivers:C:\WebDriversAcepta y cierra todas las ventanas (Aceptar en la ventana de edición, y Aceptar en la ventana de Variables de entorno).4. Verificación Final (Garantizar que Windows lo reconoce)Para confirmar que el PATH funciona, usa la línea de comandos.Abre una nueva ventana del Símbolo del sistema (CMD) o PowerShell (es fundamental que sea una ventana nueva para que cargue los cambios del PATH).Verifica Chrome: Ejecuta el siguiente comando. Si el PATH está bien, debería devolverte la versión:Bashchromedriver --version
(Resultado esperado: Versión de ChromeDriver)Verifica Edge: Ejecuta este otro comando. Debería funcionar de la misma manera:Bashmsedgedriver --version
(Resultado esperado: Versión de Microsoft Edge WebDriver)Si ambos comandos devuelven la versión, ¡felicidades! Los WebDrivers están correctamente instalados, el PATH está configurado y KNIME ya no debería tener problemas para obtener los drivers.5. Configuración en KNIMEEn tus flujos de trabajo, usa el nodo Start WebDriver. Ahora, en lugar de poner una ruta específica, simplemente asegúrate de que el campo para el "Driver file path" esté vacío. KNIME buscará el driver automáticamente en el PATH y usará el que corresponde al navegador que hayas seleccionado.
